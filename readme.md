# koishi-plugin-nats

[![npm](https://img.shields.io/npm/v/koishi-plugin-nats?style=flat-square)](https://www.npmjs.com/package/koishi-plugin-nats)

NATS 是一个简单、安全且高性能的开源数据层，专为云原生应用、物联网消息传递和微服务架构而设计。

## Special thanks

开发参考：koishi-plugin-redis

插件里的 devDependencies 是按照 https://forum.koishi.xyz/t/topic/6732/2 添加的，目的是实现插件源码推到github后自动发布到npm。
